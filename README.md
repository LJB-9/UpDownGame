# UpDownGame

## 사용한 기능
* Add Slider
* Add 'Hit Button'
* Receive value changed events from the slider
* Add 'RESET Button'
* Add labels presenting informaion
* Generate the random number
* Compare the random number with input number
* Show alerts
* Implement 'reset'feature
* Add 'Credit'view
